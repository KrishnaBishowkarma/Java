		Java:
		Java is a high-level, class-based, object-oriented programming language. It is a general-purpose
		programming language intended to let programmers write once, run anywhere (WORA). It is developed
		by James Gosling at Sun Microsystem in May 1995.
		
		
		JDK (Java Development Kit):
		JDK is a software development kit that provides necessary tools to develop Java Applications.
		Which includes Java Compiler(JavaC), Java Runtime Environment(JRE),and various development tools
		and libraries.
		
		JRE (Java Runtime Environment): 
		JRE is a runtime environment that is required to run Java applications. It includes the Java Virtual
		Machine(JVM) and core libraries necessary for executing Java programs.JRE Does not include development
		tools like a compiler.It executes Java applications by providing the JVM and necessary libraries.
		
		JVM (Java Virtual Machine): 
		JVM is a virtual machine that executes Java ByteCode. It is responsible for interpreting and executing 
		Java programs. JVM is platform-dependent and translated ByteCode into machine code specific to the 
		underlying hardware. 
		
		JavaC(Java Compiler):
		JavaC is the Java compiler, which is part of the JDK. It translates Java source code into ByteCode,
		that can be executed by the JVM. The compiler is used during the development phase to convert 
		human-readable Java code into a format that can be run on the JVM.	
		
		
		
		Primitive Data Types in Java:
		
		1. `byte`: Used to store integer values ranging from -128 to 127, occupying 1 byte of memory.
		2. `short`: Used to store integer values ranging from -32,768 to 32,767, occupying 2 bytes of memory.
		3. `int`: Used to store integer values ranging from -2^31 to 2^31 - 1, occupying 4 bytes of memory.
		4. `long`: Used to store integer values ranging from -2^63 to 2^63 - 1, occupying 8 bytes of memory.
		5. `float`: Used to store single-precision floating point numbers, occupying 4 bytes of memory.
		6. `double`: Used to store double-precision floating point numbers, occupying 8 bytes of memory.
		7. `boolean`: Used to store boolean values (`true` or `false`), typically occupying 1 bit of memory,
		 	though the actual size is implementation-dependent.
		8. `char`: Used to store a single Unicode character, occupying 2 bytes of memory.
		
		
		
		
		Key Points: 
		- JDK contains development tools (like Java Compiler) and the JRE.
		- JRE contains the JVM along with core libraries required to run Java applications.
		- JVM is the virtual machine responsible for executing Java ByteCode.
		- JShell Introduced in Java 9.
		- Java only understands ByteCode.
		- ByteCode is specially a compiled Java code followed by .class extension.
